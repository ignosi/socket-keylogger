# socket io keylogger

This is a keylogger app that logs the aroows keys and record time, users needs to login first before using the app.



### Usage

```sh
$ npm install
```

```sh
$ npm start
# Or run with Nodemon
$ npm run dev

# Visit http://localhost:5000
```

### MongoDB

Open "config/keys.js" and add your MongoDB URI, local or Atlas
